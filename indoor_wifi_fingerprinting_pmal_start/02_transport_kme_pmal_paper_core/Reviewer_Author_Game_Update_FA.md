# چرخه داور-نویسنده برای نسخه اصلاحی

## حمله داور 1: مقاله مسئله را حل نمی‌کند؛ فقط تفسیر می‌دهد.

**اصلاح:** یک مسئله مادر به شکل nested minimax وارد شد:

`min_q max_y min_h min_ytilde sup_Q J_t(...)`

این دقیقاً نشان می‌دهد انتخاب فعال، بدترین برچسب query، آموزش classifier، تکمیل برچسب‌های نامعلوم و ابهام توزیعی چگونه در یک صورت‌بندی واحد قرار می‌گیرند.

## حمله داور 2: KME و Wasserstein عوضی گرفته شده‌اند.

**اصلاح:** جایگزینی مستقیم حذف شد. اکنون مقاله یک زنجیره conservative calibration دارد:

`Wasserstein ball -> Hilbert/KME ball -> finite-dimensional TC-BKME-PMAL score`

یعنی KME به عنوان surrogate محاسباتی استفاده می‌شود، نه ادعای برابری با Wasserstein.

## حمله داور 3: آزمایش ندارید.

**اصلاح:** بسته چندفایلی Python اضافه شد. GAUSSDIST variable/equal و digit-distribution اجرا شده‌اند. 20 Newsgroups هم به صورت script جدا آماده است.

## حمله داور 4: فرمول‌ها و متن به هم می‌ریزد.

**اصلاح:** فرمول‌های عریض چندخطی شدند و PDF با render کنترل شد.
