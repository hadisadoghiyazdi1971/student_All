# خلاصه پیاده‌سازی نسخه 2

- رابط Streamlit فارسی با دو گزینه مستقل KeyBERT و Ollama
- تحلیل ترکیبی: متن + خروجی KeyBERT + کاندیدهای taxonomy به Ollama
- کنترل سقف کلمات کلیدی، حداقل امتیاز، MMR، حذف عبارات عمومی و تکراری
- Ollama با خروجی JSON ساختاریافته و اعتبارسنجی Pydantic
- تطبیق حوزه با دو فایل Excel واقعی دانشگاه و نرمال‌سازی فارسی
- بازیابی مقاله با embedding چندزبانه و امتیاز hybrid معنایی/اصطلاحی
- cache اتمیک embedding برای taxonomy و مقاله‌ها
- رتبه‌بندی افراد با ارتباط مقاله، نقش نویسنده، استناد و FWCI
- توضیح شفاف دلیل انتخاب هر فرد و هشدار نتیجه ضعیف
- پیش‌نویس اختیاری نامه با Ollama یا قالب امن محلی
- خروجی قابل ویرایش و دانلود TXT/DOCX راست‌چین
- ذخیره پایدار تنظیمات در SQLite
- OCR تصویر و PDF اسکن‌شده
- اعتبارسنجی داده، log چرخشی، Docker، تست و مستندات استقرار

## وضعیت آزمون

در زمان تحویل ۱۰ تست خودکار با موفقیت اجرا شده‌اند. تست زنده مدل واقعی به نصب محلی Ollama، دریافت `qwen3:8b` و قرار دادن CSVهای اصلی دانشگاه نیاز دارد.
