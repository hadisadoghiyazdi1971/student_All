# مسیر قدیمی

در نسخه قبلی، پروفایل استادها مستقیما توسط LLM نرمال و رتبه‌بندی می‌شد. این مسیر از جریان اصلی حذف شده است چون نسخه جدید از مقاله‌های واقعی، taxonomy دانشگاه و رتبه‌بندی قابل توضیح استفاده می‌کند.

کلاس‌های سازگاری `Professor`، `ProfessorMatch` و `RoutingResult` در `models.py` حفظ شده‌اند تا فایل‌های قدیمی در زمان مهاجرت بلافاصله دچار خطای import نشوند. این مسیر نباید در استقرار جدید فعال شود.
