من یک اجرای تست با داده‌ی مصنوعی هم انجام دادم و کد بدون خطای اجرایی کار کرد.

برای تست سریع:

pip install -r requirements.txt
python run_all.py --dataset synthetic --out results_synthetic --components 20 --latent-dim 15

برای MovieLens-100K:

python run_all.py --dataset ml100k --data-dir data/ml-100k --out results_ml100k --components 80 --latent-dim 40

برای MovieLens-1M:

python run_all.py --dataset ml1m --data-dir data/ml-1m --out results_ml1m --components 100 --latent-dim 50

برای Yelp به‌صورت CSV:

python run_all.py --dataset csv --ratings-csv data/yelp_ratings.csv --out results_yelp --components 110 --latent-dim 50

برای ساخت جدول LaTeX بعد از اجرا:

python make_latex_tables.py --results-dir results_ml100k
